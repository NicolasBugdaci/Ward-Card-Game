# Basic War Card Game

A little game designed to refresh and train on Swift and Xcode.

The code will draw a card for each player and then award a point to the user or CPU. In the event of a tie, no points are awarded.
## Lessons reviews

- String, Int
- Booleans
- Functions
- Statements
- Loops
- Conditions
- Xcode
